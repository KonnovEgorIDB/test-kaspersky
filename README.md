# User Management App

Тестовое приложение для управления списком пользователей.

## Функциональность

- Просмотр списка пользователей
- Добавление пользователя
- Удаление пользователя
- Сортировка списка
- Сохранение данных в localStorage

## Технологии

- React
- Vite
- React Router DOM
- Context API
- CSS Modules

## Инструменты качества кода:

- ESLint
- Prettier
- Stylelint
- Lefthook (git hooks)

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```
