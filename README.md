# User Management App

Тестовое приложение для управления списком пользователей.

## Функциональность

- Просмотр списка пользователей
- Добавление пользователя
- Удаление пользователя
- Сортировка списка
- Сохранение данных в localStorage

## Технологии

- React
- Vite
- React Router DOM
- Context API
- CSS Modules

## Инструменты качества кода:

- ESLint
- Prettier
- Stylelint
- Lefthook (git hooks)

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

## Выводы по проектированию UI

- **Страница списка пользователей** была разработана самостоятельно. В них демонстрируются навыки проектирования вручную: построение таблицы пользователей, модального окна для добавления пользователей.

- **Страница групп пользователей** была сгенерирована с помощью LLM. Это демонстрирует умение использовать большие языковые модели для ускорения проектирования UI, при этом дорабатывая интерфейс для корректного UX.
